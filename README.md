# bingoslackbot

Bingo! is a simple Slack Bot with a couple of useful commands.

Currently, there are six commands:

- /bingo-help: lists all currently available commands

- /bingo-ping: grabs the current ping of the bot

- /bingo-catfact: fetches a catfact

- /bingo-joke: fetches a joke

- /bingo-forecast: fetches the current weather in a specific location given a U.S. zip code

- /bingo-colorpallete: fetches color palletes based on the word you give to Bingo!
